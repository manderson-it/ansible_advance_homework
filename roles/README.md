Overview of roles
=========

The ./roles folder in this repository contains all Ansible roles required.
No roles need to be pulled from Galaxy.

```bash
.
├── app-tier
├── base-config
├── config-tower
├── db-tier
├── lb-tier
├── osp-facts
├── osp-instance-delete
├── osp-servers
├── osp-setup
└── setup-workstation
```
